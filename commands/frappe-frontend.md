# Frappe Frontend

Handle the task as Frappe frontend work.

- First choose the right frontend surface: desk-native, `www`, `Web Page`, `Workspace`, Vue/`frappe-ui`, React, or external SPA.
- Prefer Vue when the project already uses `frappe-ui` or desk-native Vue patterns.
- Prefer React when the project is already React-based or clearly external-SPA oriented.
- Do not flatten all frontend work into one recommendation.
