# Frappe Customization

Handle the task as Frappe customization-surface work.

- Choose the right surface before suggesting code.
- Consider `Custom Field`, `Property Setter`, `Client Script`, `Server Script`, `Workspace`, `Web Page`, `Page`, `Print Format`, `Report`, `Dashboard`, `Workflow`, `Notification`, `Webhook`, and related admin DocTypes.
- Prefer built-in configuration and metadata surfaces before hooks or invasive code changes.
